# Sample Login Credentials (Demo)

Use these for **demo login** when Microsoft SSO is not configured. On the login page, click **"Use demo login (sample credentials)"** and enter one of the pairs below.

---

## Student

| Field     | Value                  |
|----------|-------------------------|
| Username | `student123`            |
| Password | `SecureStudentPass1!`   |
| Role     | Student                 |

After login you are redirected to the **Student Dashboard**. You will provide what should be displayed on the student side; the dashboard currently shows a placeholder.

---

## Instructor (optional)

| Field     | Value                  |
|----------|-------------------------|
| Username | `instructor1`           |
| Password | `InstructorPass1!`      |
| Role     | Instructor              |

---

## Admin (optional)

| Field     | Value                  |
|----------|-------------------------|
| Username | `admin1`                |

| Role     | Admin                   |

---

## Microsoft Sign-in

When OAuth is configured (`NEXT_PUBLIC_API_URL` and backend auth are set up), the main **"Sign in with Microsoft"** button redirects to Microsoft for SSO. Until then, use the demo login and sample credentials above.
